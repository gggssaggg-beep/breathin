# Голосовые подсказки фаз (П8)

Сгенерированы 2026-07-18 нейросинтезом Microsoft (edge-tts 7.2.8):
ru — голос `ru-RU-DmitryNeural`, en — `en-US-ChristopherNeural`;
темп −10%, медленные варианты (`*_slow.wav`, для фаз ≥ 6 с) — −25%.
Выбор голоса и правило темпа — решение владельца («если вдох или выдох
долгий — медленная версия, чтоб спокойствие считывалось»).

Обработка (ffmpeg 8.1): silenceremove с обоих концов, фейды 12/35 мс,
loudnorm I=-17 TP=-2, ресемпл 44 100 Гц mono s16 — тот же SR, что у
остальных наборов звука.

Файлы: `inhale/exhale/hold/prep(.wav)` + `inhale_slow/exhale_slow`.
`hold` используется для обеих задержек (после вдоха и после выдоха).

Примечание о происхождении: edge-tts — неофициальный доступ к голосам
Microsoft Edge (read-aloud); формальной лицензии на встраивание нет,
использование в бесплатном приложении согласовано владельцем 2026-07-18.
Перегенерация: `python -m edge_tts --voice ru-RU-DmitryNeural --rate=-10% --text "Вдох" --write-media inhale.mp3` → обработка выше.
