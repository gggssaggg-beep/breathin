# Генерация мира «Дыши» (скилл scroll-world)

Лендинг-«пролёт»: скролл скрабит непрерывный полёт камеры сквозь 6 глиняных диорам.
Архитектура B (дайв в сцену + аэро-коннектор) — мир миниатюрный, «карта островов».
Страница и движок уже готовы (`../index.html`, `../scrub-engine.js`); осталось
сгенерировать ассеты через Higgsfield.

## Маршрут (6 сцен)

| # | id | Сцена | Секция лендинга |
|---|----|-------|-----------------|
| 1 | `dawn` | Рассвет над долиной, павильон у озера | Интро: «Спокойствие — в одном вдохе» |

Палитра = приложение: бирюза `#3E7C86`, фон `#EEF4F3`, янтарь `#F9A825`,
шалфей `#7FA88C`, голубой `#3F7FBF`, лаванда `#9C6BD6`.
Стилевая преамбула вшита в каждый `still_*.txt` ДОСЛОВНО — не редактировать частично.

## Порядок запуска (после `higgsfield auth login` и утверждения бюджета)

```bash
cd landing/generation
# 0. калибровка цены: снять баланс, сделать ОДНУ картинку и ОДНО видео, снять снова
higgsfield workspace list
# 1..6 — стадии (каждая генерация 3–8 мин, запускать фоном):
bash pipeline.sh stills   # 6 картинок; ПОТОМ ГЛАЗАМИ проверить цельность стиля!
bash pipeline.sh webp     # постеры в ../assets
bash pipeline.sh dives    # 6 дайвов (VMODEL=seedance_2_0 по умолчанию)
bash pipeline.sh frames   # граничные кадры из отрендеренных дайвов
bash pipeline.sh conns    # 5 коннекторов — швы держатся на кадрах дайвов
bash pipeline.sh encode   # crf20 g8 faststart → ../assets/vid
```

Драфт-тир: `VMODEL=seedance_2_0_mini bash pipeline.sh dives` (и `conns`).
Переролл одного клипа: `bash pipeline.sh dive wimhof`, `bash pipeline.sh conn 3`.
NSFW-фальстарт (Seedance любит флажить интерьеры/фигуры): переролл 2–3 раза →
зачистить слова-триггеры → тот же клип на `VMODEL=kling3_0` с теми же кадрами →
крайний случай: в `index.html` поставить `null` вместо коннектора.

## QA перед публикацией (SKILL Step 8)

- скриншоты до/после каждого шва — кадры должны совпадать;
- `video.seekable.end(0) > 0` (blob работает), currentTime следует скроллу;
- reduced-motion → статичные постеры;
- телефонный вьюпорт: постеры видны, ничего не наезжает.

Сырьё (`work/`) в git не идёт (см. .gitignore); перекодированные клипы в
`../assets/vid` — идут (≈8 МБ × 11).
